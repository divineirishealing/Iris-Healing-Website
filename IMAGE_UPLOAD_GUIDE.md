## 🖼️ NEW FEATURE: Direct Image Upload!

You can now upload images directly from your computer instead of using URLs!

## 📸 How to Upload Images

### Step 1: Login to Admin Panel
Go to: https://iris-staging.preview.emergentagent.com/admin
- Username: `admin`
- Password: `divineadmin2024`

### Step 2: Choose Your Upload Method

When adding or editing a Program or Session, you'll see two buttons:

**Option 1: Upload Image** (Upload from your computer)
- Click the "Upload Image" button
- You'll see a drag-and-drop area
- Click on it OR drag your image file onto it
- Image will upload automatically
- Preview appears once uploaded

**Option 2: Use URL** (Use image from internet)
- Click the "Use URL" button
- Paste any image URL
- Click "Set URL"

## ✅ Supported Image Types
- JPG / JPEG
- PNG
- GIF

## 📏 Image Requirements
- **Maximum file size:** 5MB
- **Recommended size:** 600x400 pixels or larger
- **Format:** Landscape orientation works best

## 🎯 Quick Steps to Add a New Program with Your Image

1. **Go to Admin Panel** → Programs Tab
2. **Fill in the form:**
   - Title: Enter program name
   - Category: e.g., "Foundation", "Healing"
   - Description: Write details
3. **Upload Your Image:**
   - Click "Upload Image" button
   - Click the upload area
   - Select image from your desktop
   - Wait for upload (takes 2-5 seconds)
   - See preview appear
4. **Click "Add Program"**
5. **Done!** Your program appears on the website instantly

## 🔄 To Change an Existing Image

1. Click the **Edit** button (pencil icon) on any program/session
2. You'll see the current image
3. Click the **X button** on the image to remove it
4. Upload a new image OR paste new URL
5. Click "Update Program/Session"

## 💡 Pro Tips

### Best Practices:
✅ Use high-quality, clear images
✅ Keep file size under 2MB for faster loading
✅ Use horizontal (landscape) images for best fit
✅ Choose calming, professional images for healing content

### Common Issues:

**Image not uploading?**
- Check file size (must be under 5MB)
- Make sure it's an image file (.jpg, .png, etc.)
- Try a different browser if issues persist

**Image looks blurry?**
- Use higher resolution images (at least 1200x800 pixels)
- Avoid very compressed images

**Want to use image from internet?**
- Switch to "Use URL" mode
- Right-click on any image online → "Copy Image Address"
- Paste the URL

## 🆘 Troubleshooting

**Problem:** "File too large" error
**Solution:** 
- Compress your image using: https://tinypng.com
- Or choose a smaller image

**Problem:** Upload button not working
**Solution:**
- Refresh the page (F5)
- Clear browser cache
- Try different browser (Chrome, Firefox)

**Problem:** Image shows but looks broken on website
**Solution:**
- Make sure image uploaded completely
- Try re-uploading
- Check image isn't corrupted
