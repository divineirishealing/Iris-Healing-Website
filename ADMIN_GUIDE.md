## 🎯 How to Use the Admin Panel

### Step 1: Login
1. Go to: https://divine-healing-admin.preview.emergentagent.com/admin
2. Enter username: `admin`
3. Enter password: `divineadmin2024`
4. Click "Login"

### Step 2: Navigate
You'll see 5 tabs at the top:
- **Programs** - Manage flagship programs
- **Sessions** - Manage personal sessions
- **Testimonials** - Manage video testimonials
- **Stats** - Update website statistics
- **Subscribers** - View newsletter subscribers

### Step 3: Add Content

#### To Add a Program:
1. Click "Programs" tab
2. Fill in the form on the left:
   - Title: "Your Program Name"
   - Category: "Foundation" or "Healing" etc.
   - Description: Detailed description
   - Image URL: Copy from Unsplash or your hosting
3. Click "Add Program"
4. Your program appears instantly on the right

#### To Edit a Program:
1. Click the pencil icon (✏️) next to any program
2. The form fills with current data
3. Make your changes
4. Click "Update Program"

#### To Delete a Program:
1. Click the trash icon (🗑️) next to any program
2. Confirm deletion
3. It's removed immediately

### Step 4: Finding Images

**Free Image Sources:**
- Unsplash: https://unsplash.com
- Pexels: https://pexels.com

**How to get image URL:**
1. Find an image you like
2. Right-click and select "Copy Image Address"
3. Paste into the Image URL field

**Example URLs:**
```
https://images.unsplash.com/photo-1545389336-cf090694435e?w=600&h=400&fit=crop
https://images.unsplash.com/photo-1506126613408-eca07ce68773?w=600&h=400&fit=crop
```

### Step 5: YouTube Testimonials

**Finding YouTube Video ID:**
From URL: `https://www.youtube.com/watch?v=i3UJ6t6xYwQ`
Video ID is: `i3UJ6t6xYwQ` (the part after `v=`)

**To Add:**
1. Go to "Testimonials" tab
